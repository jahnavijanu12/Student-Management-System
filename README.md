# Student Management System

A Java console-based application to manage student records using OOP and file handling.

## Features
- User Login
- CRUD Operations
- File Handling
- Sorting
- Dashboard
- Report Export
- CSV Export

## Technologies
- Java
- OOP
- Collections
- File Handling

## Project Structure
Main.java
Student.java
StudentService.java
FileManager.java
LoginService.java 

## Run
Compile:
javac *.java

Run:
java Main
